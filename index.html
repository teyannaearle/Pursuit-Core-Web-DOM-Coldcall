Cold Caller

Welcome to a coldcall app! To complete the app, finish the functions in main.js

When the 'Get Next Up' button is hit:

  1. The next name on the list should appear in the span with id="next-fellow"
  2. The first time the button is hit, the first name on the list should appear.
  3. If we've exhausted the list, the span with id="next-fellow" should display 'Please Shuffle Class!'

When the Shuffle Class button is hit:

  1. The order of names should be shuffled randomly
  2. After hitting the Shuffle button, the 'Get Next Up' button should start at the beginning of the list of names.
